Question-1: What are the different ways to select an element in the DOM?

document.getElementById("id"). Selects a specific element by its ID. Returns only one element. document.getElementsByClassName("class")Selects all elements with the given class name. Returns an HTMLCollection (like a list). document.getElementsByTagName("tag")Selects elements by tag name ("p", "div"). Returns multiple elements. document.getElementsByName("name") Selects elements by the name attribute. Often used with form elements. document.querySelector("selector") Selects the first matching element using a CSS selector. Example: "#id", ".class". document.querySelectorAll("selector") Selects all matching elements using a CSS selector.Returns a NodeList (like a list).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML-Gets or sets HTML content inside an element. Can read/write tags and text. Example: element.innerHTML = "<p>Hello!!!</p>"; innerText-Gets or sets the visible text (what user sees). Ignores hidden elements (display: none). Example: element.innerText = "Hello"; textContent-Gets or sets all text, even if hidden. Faster than innerText. Example: element.textContent = "Hello";

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you handle events (like click) at a parent element, instead of adding listeners to each child. Why use it? Better performance.Fewer event listeners.Works for new (dynamic) child elements. How it works: Add one event listener to the parent.Use event.target to find which child triggered the event

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event happens on a child element and then moves up to its parent, grandparent, and so on. Use: Useful for event delegation.Useful for event delegation

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create Element: let newDiv = document.createElement("div"); 2. Add Element: parent.appendChild(newDiv); 3. Remove Element: element.remove();
